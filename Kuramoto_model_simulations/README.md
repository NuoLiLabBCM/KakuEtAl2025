# KakuEtAl2025
Kuramoto model simulation code for [**Kaku, Liu et al. A brainstem map of orofacial rhythms**](https://www.biorxiv.org/content/10.1101/2025.01.27.635041v1)


## Simulate a toy Kuramoto model with two oscillators, testing a range of coupling strength:

run sim_simple_asym_param_range.m


## Simulate a simple Kuramoto model with intrinsic frequencies mimicking empirical data

run sim_simple_asym_real.m


## Compare a event-gated model vs. a continuously-coupled Kuramoto model

run sim_model_comparison_stall_conditional_phase.m

run sim_model_comparison_kuramoto_conditional_phase.m
